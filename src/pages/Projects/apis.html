• API'S
    • Questions and response

    This project is an api in node.js, which can be consumed in different front ends. It is done using the mysql database and aims to create questions and answers and show them.

    • APP STORE

    This is a store sales and purchase control api. Contains a complete CRUD and authentication.

    • Product registration

    This api registers products and accepts uploads of images. Multer was used for the images and mysql to register these products in the database.

    • DOCKER

    In this api you will learn how to use the docker in your development environment.

    • PAGINATION

    In this api I teach you how to create pages in node.js.

    • TDD

    In this api you will learn how to create automated tests in node.js with jest. API under development
